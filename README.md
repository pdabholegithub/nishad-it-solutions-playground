# Nishad IT Playground 🧪

An elite-level, AI-powered automation practice platform designed for professional QA Engineers and SDETs. Built with **React 18**, **TypeScript**, and **Vite**.

🌐 **Live Playground:** [pdabholegithub.github.io/nishad-it-solutions-playground](https://pdabholegithub.github.io/nishad-it-solutions-playground/)  
🏛️ **Main Institute:** [nishad-institute.vercel.app](https://nishad-institute.vercel.app/)

---

## 🚀 Key Modules & Features

### 1. API Mastery Lab (Complete Journey)
A specialized environment for practicing complex API workflows (Postman, Playwright, Rest Assured):
- **API Chaining**: Authentication ➡️ Creation ➡️ Verification ➡️ Cleanup lifecycle.
- **Live Traffic Log**: Real-time request/response capture with status codes and payloads.
- **Stateful Mock Server**: A persistent "Database" that updates as you run your tests.
- **Negative Testing**: Built-in endpoints for 401 Unauthorized, 404 Not Found, and 400 Bad Request.

### 2. Elite Mastery Challenges
High-complexity automation scenarios rarely found on other practice sites:
- **Deep Shadow DOM**: Nested shadow roots for advanced selector practice.
- **Race Conditions**: Time-delayed rendering for synchronization & wait-logic.
- **SVG Mastery**: Interactive vector path targeting and verification.
- **Download Mastery**: Triggering and asserting local file content in automation.
- **Window Management**: Multiple tab/window switching and context control.
- **Sortable Lists**: Drag & Drop reordering logic.

### 3. Advanced UI Controls
High-density components to test robust automation scripts:
- **Dynamic Tables**: High-density data with horizontal scrolling, sticky columns, and pagination.
- **Form Controls**: Star ratings, native color pickers, and multi-step wizards.
- **Interactions**: Context menus (right-click), double-clicks, and keyboard events.
- **Autocomplete**: Debounced search with dynamic suggestion lists.

### 4. AI-Powered Assistant
- **Gemini 2.0 SDET Chat**: A built-in AI assistant specialized in writing Playwright and Rest Assured code, debugging scripts, and explaining automation patterns.

## 🎨 Design System
- **Theme**: Sunset Amber & Royal Indigo (High-contrast professional palette).
- **Aesthetics**: Glassmorphism, smooth micro-animations, and responsive layouts.
- **Fonts**: Outfit (Headings) and Inter (Body).

## 🛠️ Tech Stack
- **Framework**: React 18 + Vite
- **Language**: TypeScript
- **Icons**: Lucide React
- **Styling**: Tailwind CSS + CSS Variables
- **API Simulation**: LocalStorage Persistence + State Management

---

## 📖 How to Use
1. **Explore**: Navigate through the sidebar to find specific controls (Forms, API, Mastery).
2. **Automate**: Write your automation scripts against the `data-testid` attributes provided.
3. **Verify**: Use the "Live Logs" or "Database" views on each page to verify your script's impact on the application state.
